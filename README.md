# KNN Classification Project

This project applies the K-Nearest Neighbors (KNN) algorithm to classify data based on given features.

## 📌 Project Overview
The goal of this project is to build a machine learning model that predicts the target class using the KNN algorithm. The workflow includes data exploration, preprocessing, model training, evaluation, and tuning.

## 🧪 Steps Performed
- Loaded and explored the dataset
- Visualized data using pairplots (EDA)
- Standardized features using StandardScaler
- Split data into training and testing sets
- Trained a KNN model
- Evaluated performance using confusion matrix and classification report
- Tuned the model by selecting the optimal K value

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

## 📊 Results
The model performance improved after selecting a better K value, achieving higher accuracy and balanced precision/recall.
